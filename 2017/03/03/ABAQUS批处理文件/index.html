      ABAQUS批处理文件

      标签： ABAQUS 批处理

      ABAQUS批处理能够对一个或多个inp文件进行提交计算，而批处理文件的建立也非常简单。
      首先，在inp文件所在文件夹建立一个start.txt的文本文档。如我的inp文件名分别为job-1.inp, job-2.inp。
      其次，对start.txt文档进行编辑，内容如下:

      call abaqus job=job-1 cpus=4 int
call abaqus job=job-2 cpus=4 int
pause

      最后，讲start.txt的后缀改为bat 双击运行即可。

      其中int 为interactive，可实现多个job依次运行，去掉int的话多个job同事运行。pause 目的是在计算完成或者出错以后黑窗不被马上关闭，可以看到出错的原因。当然，即便是黑窗被关掉我们也可以通过查看与任务同名的.dat，.sta，.mag文件查看计算过程中出现警告和错误。
      另外在计算过程中可以随时打开.odb文件查看计算进度。

      ps:对于abaqus不是内部命令问题的解决可以将abaqus.bat文件的路径添加到系统环境路径即可。
